bool ohci_probe ( uintptr_t  base_addr,
usb_hcd_t *  hcd 
)

Initialises the OHCI device at the specified base address, probes all the attached USB devices, and configures any HID USB keyboard devices it finds to generate periodic interrupt transfers that report key presses.

If successful, initialises the specified host controller driver object accordingly.

Parameters
base_addr- the base address of the device in virtual memory
hcd- a pointer to a pre-allocated host controller driver object that can be used for this device
Returns
true if the device was successfully initialised and one or more keyboards were found, otherwise false.
