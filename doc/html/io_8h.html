Detailed Description

Provides macro definitions for the x86 IO instructions inb/inw/inl/outb/outw/outl and the "string versions" of the same (insb/insw/insl/outsb/outsw/outsl).

You can also use "pausing" versions of the single-IO instructions (inb_p/inw_p/..).

This file is not meant to be obfuscating: it's just complicated to (a) handle it all in a way that makes gcc able to optimize it as well as possible and (b) trying to avoid writing the same thing over and over again with slight variations and possibly making a mistake somewhere.
