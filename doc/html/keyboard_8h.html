char get_key ( void  )

Checks if a key has been pressed and returns the primary ASCII character corresponding to that key if so, otherwise returns the null character.

F1 to F10 are mapped to the corresponding decimal digit (F10 -> 0). All other keys that don't have a corresponding ASCII character are ignored. Characters are only returned for key presses, not key releases. A US keyboard layout is assumed (because we can't easily do anything else).
