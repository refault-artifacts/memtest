Returns an unsigned numeric value entered on the keyboard. 

Echoes the input to the display field located at (row,col), limiting it to field\+\_\+width characters. If the entered value is prefixed by \char`\"{}0x\char`\"{}, assumes base 16, otherwise assumes base 10. If the value is suffixed by \textquotesingle{}K\textquotesingle{}, \textquotesingle{}P\textquotesingle{}, \textquotesingle{}M\textquotesingle{}, \textquotesingle{}G\textquotesingle{}, or \textquotesingle{}T\textquotesingle{}, the returned value will be scaled by 2$^\wedge$10, 2$^\wedge$12, 2$^\wedge$20, 2$^\wedge$30, or 2$^\wedge$40 accordingly. The returned value will also be scaled by 2$^\wedge$shift.
