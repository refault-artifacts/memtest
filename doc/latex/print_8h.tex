Emulates the standard printf function. 

Printing starts at location (row,col).

The conversion flags supported are\+:
\begin{DoxyItemize}
\item left justify 0 pad with leading zeros S add space between number and unit (k specifier only)
\end{DoxyItemize}

The conversion specifiers supported are\+: c character (int type) s string (char$\ast$ type) i signed decimal integer (int type) u unsigned decimal integer (uintptr\+\_\+t type) x unsigned hexadecimal integer (uintptr\+\_\+t type) k K$<$unit$>$ value (scaled to K/\+M/\+G/T) (uintptr\+\_\+t type)

The only other conversion option supported is the minimum field width. This may be either a literal value or \textquotesingle{}$\ast$\textquotesingle{}.

Returns the next column after the formatted string. \mbox{\Hypertarget{print_8h_a9ddb189ef43bfb2e4abd4d9cd0464048}\label{print_8h_a9ddb189ef43bfb2e4abd4d9cd0464048}}
